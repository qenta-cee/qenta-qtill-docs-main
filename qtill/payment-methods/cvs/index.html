CVS Gift Card

CVS Gift Card

CVS Pharmacy is a gift card payment option that is added by paysafecard.

Commands

CVS gift card uses the same commands as paysafecard. Visit paysafecard to see details!

INITIATEPAYMENT

CVS gift card payment method can be initialized if the merchant sends CVSPHARMACY for parameter INITTYPE, and PAYSAFECARD for PAYMENTTYPE.
Go to INITIATEPAYMENT of Paysafecard, and apply these changes!

Order Flow Diagram

To interpret this diagram, the successful completion of a NEW PAYMENT is assumed. Starting from this operation it’s possible to carry out different commands.

For an explanation of the different status descriptions included in this diagram have a look at payment.<n>.<m>.state and order.<n>.state which are response parameters of command GETORDERDETAILS.
