Crypto

Crypto

QENTA supports cryptocurrencies and is offered by Salamantex. With Salamantex, digital currencies such as Bitcoin can be used digitally in a risk-free and easy way.

Salamantex is managing cryptocurrency wallets for our merchants. If a merchant wants to get paid with a cryptocurrency, Salamantex creates a new address for the payment and displays the address to the buyer. Periodically checks the address for new funds and reports status changes back to the buyer and merchant.

Currently, supported cryptocurrencies are Bitcoin, Litecoin, Charry, Dash, Ethereum, Ripple, Stellar and Nimiq.

Initialization
Crypto payment method can be initialized if the merchant sends CRYPTO for parameter INITTYPE.

  1. Send a request.

    Example of Request for Initiation
    COMMAND=INITIATEPAYMENT&
MERCHANTKEY=43gADp5.UOZxPTet6O8o8K6o-7RefQB3BO21OMu4chB65LJUI_&
INITTYPE=CRYPTO&
PAYMENTTYPE=CRYPTO&
ORDERNUMBER=5316774&
LANGUAGE=en&
AMOUNT=1&
CURRENCY=EUR&
ORDERDESCRIPTION=Test&
APPLICATIONTYPE=S&
SERVICEURL=http://yourimprint.com/&
SUCCESSURL=https://yoursuccessurl.com&
CONFIRMURL=https://yourfailureurl.com&
FAILUREURL=https://yourfailureurl.com&
CANCELURL=https://yourfailureurl.com&

  2. The response looks like this:

    Example of Response for Initiation
    header=Content-Type%3A+text%2Fhtml%0D%0A&
body=%3Chtml%3E%3Chead%3E%3Cmeta+http-equiv%3D%22refresh%22+content%3D%220%3BURL%3Dhttps%3A%2F%2Fmy.test.salamantex.com%2Fapi%2Fpayment%2F9bdc07b6-f7d7-4465-9299-2fca2338e20c%22%3E%3C%2Fhead%3E%3C%2Fhtml%3E&
version=qtillserver+3.23.0.47&
status=0

  3. Decode the text with URL Decoder. Use Notepad ++ or some other URL Decoder. After decoding the response should look like this.

    header=Content-Type: text/html
&body=<html><head><meta http-equiv="refresh" content="0;URL=https://my.test.salamantex.com/api/payment/9bdc07b6-f7d7-4465-9299-2fca2338e20c"></head></html>&version=qtillserver 3.23.0.47&status=0

  4. Beautify code (delete everything above and below html tag, and replace every "+" with empty space).

    <html>
  <head>
    <meta http-equiv="refresh" content="0;URL=https://my.test.salamantex.com/api/payment/9bdc07b6-f7d7-4465-9299-2fca2338e20c">
  </head>
</html>

  5. Execute the HTML, and the Crypto mask should be seen in a browser. Enter specific test data for Crypto payment method. After that, you will be redirected to the successurl (which is invalid in this example).
