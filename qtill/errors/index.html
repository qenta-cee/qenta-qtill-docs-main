Error Messages

Error Messages

QTILL knows two types of error messages:

  • Simple

  • Compound

In the case of compound error messages, the Error Messages number and the Error Parameters number are simply added together.

Example: 11115 means: The "Card number" parameter (i.e. 115) was not specified.

The error message then reads: "Card number missing."

In the description, a {0} appears as a placeholder for the parameter text. In the {1} placeholder, the invalid parameter value is specified. The latter will be cleaned if the content is sensitive.
