Pre-Transaction Commands

Pre-Transaction Commands

PING-Checking the connection to the Qenta Payment Server

This operation checks the connection to the Qenta Payment Server.

Request Parameters

Example of Request Parameters
MERCHANTKEY=43gADp5.UOZxPTet6O8o8K6o-7RefQB3BO21OMu4chB65LJUI_&
COMMAND=PING&
LANGUAGE=en&
APPLICATIONTYPE=S&

Response Parameters

Example of Response Parameters
version=qtillserver+3.23.0.47&
status=0

CREATEORDER

This operation generates a unique order number.

Request Parameters

Example of Request Parameters
MERCHANTKEY=43gADp5.UOZxPTet6O8o8K6o-7RefQB3BO21OMu4chB65LJUI_&
COMMAND=CREATEORDER&
APPLICATIONTYPE=S&
LANGUAGE=en

Response Parameters

Example of Response Parameters
orderNumber=1181521& (1)
version=qtillserver+3.23.0.47&
status=0
1 Unique order number

GETINSTALLMENTPLANS

This command is used for getting installment plans for the payment method Riverty-Installment.
